TO USE THIS SOFTWARE:
  1. First, configure ProPresenter to enable the network settings which allows Presentation Bridge to access the data. You can use any port numbers you want, provided they are not already in use. The port for "Enable Network" is the port Presentation Bridge will use.

    Make note of this port, as you will need to enter it on the Bridge page in order to connect.

  2. Go to the Bridge page on the server running Presentation Bridge: http://127.0.0.1/bridge
    It runs on Port 80 by default.

  3. Enter in the IP address, port, control password, and library path of your ProPresenter machine.
    The library path must be the full absolute path to your ProPresenter library, as accessed from that machine.
    Example: /Users/technician/Documents/ProPresenter6/

  4. Click "Connect to ProPresenter". If you've connected successfully, the interface will begin downloading the ProPresenter data.

  5. Select a Bridge from the Bridge Config box. If it requires a password, you will need to enter it.

  6. Click "Connect to Bridge" to connect. If a Bridge is already in use, you will receive an error message.

  7. Clients can connect by going to your server IP address with these listeners:
    • / - the default "text listener"
    • /image - the image listener
    • /stagedisplay - the stage display version that shows the current and next slides
